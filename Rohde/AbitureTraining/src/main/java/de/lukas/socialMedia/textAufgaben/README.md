# Social Media Text Exercises

Small set of classes used for written tasks and quick OOP exercises related to the social media domain.

## What is included
- `Beitrag`, `Nutzer`, `DateTime` - lightweight model used in the tasks
- `SocialMediaManager` - minimal console demo
- Reuses `Bild`, `Text`, and `List` from `de.lukas.socialMediaGPT`

## Run
```bash
./gradlew classes
java -cp build/classes/java/main de.lukas.socialMedia.textAufgaben.SocialMediaManager
```

## Notes
- `Aufgabe1.md` contains the written answer (German).
